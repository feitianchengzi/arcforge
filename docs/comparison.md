# Comparison

SkillOps is not a registry, marketplace, or agent runtime.

| Category | Examples | SkillOps stance |
|---|---|---|
| Public skill directories | skills.sh, Skillery | Use them as distribution channels, not replacements for GitHub source. |
| Multi-agent sync tools | skillshare, agpack | Integrate with them instead of rebuilding every target adapter. |
| Desktop skill managers | SkillsGate, Anvil, YouSkill | Compete only where workflow governance matters: audit, profiles, drift, publish planning. |
| Private registries | SkillHub, SkillReg | Avoid hosted registry complexity for the MVP. GitHub repo access is enough for small teams. |

The core product bet is that personal and small-team skills need lifecycle tooling, not only installation tooling.
