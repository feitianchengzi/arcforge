# Contributing

Thanks for helping improve SkillOps.

## Development

```bash
npm install
npm run dev
```

Run checks:

```bash
npm run check
npm test
```

## Pull Requests

- keep changes focused
- include screenshots for UI changes
- update docs when changing user-facing behavior
- add or update tests for core logic
- avoid introducing hosted services into the MVP without discussion

## Project Direction

SkillOps should stay local-first and GitHub-first until the MVP proves demand. Prefer orchestration of existing tools over reimplementing agent-specific sync behavior.
