# Skills - 交互规范

## 交互策略

### 核心任务

用户在 Skills 页面浏览、筛选、编辑和保存当前项目中的 Skill 文件，并将项目中自然产生的 Skill 归并到另一个 Skill 项目。

### 主路径

页面左侧展示技能文件树，右侧展示当前文件内容。用户选择技能后查看文件正文和保存状态。用户编辑后保存，系统写入文件并重新扫描。

归并入口放在目录列表右上角。用户点击后选择目标 Skill 项目、本次归并的 Skill、目标项目内写入目录、目标项目配置组和当前项目应用目标目录。系统先生成计划；无冲突且用户确认后，系统把选中的 Skill 写入目标 Skill 项目，并在当前项目保存应用关系。

### 决策点

用户决定打开哪个文件、是否只看配置组范围内的技能、是否保存草稿、是否在独立窗口打开文件、是否执行归并、选择哪个目标 Skill 项目、选择哪些 Skill、写入目标项目的哪个目录和配置组。系统通过脏状态、保存状态、归并计划和冲突清单防止误操作。

### 信息揭示

文件树展示技能目录和文件。编辑区展示文件内容、保存状态和独立窗口入口。归并面板展示目标 Skill 项目、Skill 选择、目标目录、配置组、应用目标目录、计划结果和冲突清单。

### 状态流

无选择时显示引导。选择文件后显示编辑器。修改后进入未保存状态。保存成功后刷新状态。保存失败时保留编辑内容和错误原因。归并前展示计划。目标目录中存在内容不同的同名 Skill 时停止执行并展示冲突。归并成功后刷新当前项目应用关系。

### 反馈与恢复

切换文件前若存在未保存修改，系统要求保存、放弃或取消。保存失败不丢失编辑内容。归并失败保留目标 Skill 项目、Skill 选择、目标目录和配置组。远程项目下载失败保留地址。冲突不会自动覆盖，用户修复后重新生成计划。

### 输入输出边界

输入包括文件选择、配置组筛选、文本编辑、保存、放弃、独立窗口打开、目标 Skill 项目选择、本地目录输入、远程项目输入、Skill 选择、目标目录、配置组、应用目标目录和归并确认。输出为文件内容、保存结果、扫描结果、归并计划、写入结果、目标项目配置组变更和应用关系。

## 页面状态

| 状态 | 触发条件 | data-kit 关键控件 |
|------|---------|-----------------|
| 文件列表 | 进入 Skills 页面 | SplitView / FileTree |
| 编辑中 | 用户选择技能文件 | TextEditor |
| 未保存 | 用户修改内容 | DirtyIndicator |
| 保存失败 | 文件写入失败 | InlineError |
| 无技能 | 当前项目无技能 | ContentUnavailableView |
| 归并目标 | 用户点击目录列表右上角归并入口 | Sheet / ProjectPicker |
| 归并计划 | 用户选择目标项目、Skill、目录和配置组 | PlanPreview |
| 归并冲突 | 目标项目目录存在内容不同的同名 Skill | ConflictList |
| 归并结果 | 归并完成或失败 | ResultView |

## 交互行为

- **配置组筛选:** 用户选择配置组后文件树只显示命中技能。
- **编辑保存:** 用户保存后系统写入文件并重新扫描该技能。
- **切换保护:** 存在未保存修改时，切换文件触发确认。
- **独立窗口:** 用户点击 Open in window 后系统打开独立文件编辑窗口，并带入当前配置组过滤和选中文件。
- **选择目标项目:** 用户在归并面板中选择本地目录或输入远程仓库；远程仓库先下载到本地缓存，再作为本地 Skill 项目使用。
- **选择归并范围:** 用户选择当前项目 Skill、目标项目写入目录、目标项目配置组和当前项目应用目标目录。
- **生成计划:** 系统比较当前项目 Skill 与目标项目目录，输出新增、一致和冲突清单。
- **执行归并:** 用户确认无冲突计划后，系统写入目标项目、合并目标项目配置组，并记录当前项目应用关系。

## 弹窗/对话框

| 名称 | 触发 | data-kit | 内容 |
|------|------|----------|------|
| Unsaved changes | 切换文件或离开页面 | ConfirmationDialog | 保存、放弃、取消 |
| Save error | 保存失败 | InlineError | 错误原因和重试 |
| Merge skills | 点击目录列表右上角归并入口 | Sheet | 目标 Skill 项目、Skill、目录、配置组和应用目标目录 |
| Conflict list | 归并计划发现内容冲突 | Sheet | 冲突路径、当前项目路径、目标项目路径和重新计划入口 |

## 错误处理

| 错误类型 | 展示方式 | 恢复操作 |
|---------|---------|---------|
| 文件不存在 | 编辑器错误态 | 重新扫描 |
| 写入失败 | 编辑器顶部错误 | 重试或复制内容 |
| 编码不可读 | 只读错误态 | 外部打开 |
| 远程项目下载失败 | 归并 Sheet 错误 | 修改地址后重试 |
| 目标项目不可写 | 计划错误 | 更换目标项目或目标目录 |
| 同名内容冲突 | 冲突清单 | 修复后重新计划 |
| 应用关系保存失败 | 结果错误 | 查看详情后重试 |

## 加载策略

| 场景 | 策略 |
|------|------|
| 文件加载 | 保留文件树选择，编辑区显示加载 |
| 保存 | 禁用保存按钮，保留可见内容 |
| 扫描刷新 | 局部刷新技能状态，不清空编辑器 |
| 归并计划 | 禁用执行按钮，保留所有选择 |
| 执行归并 | 显示文件级写入状态 |
