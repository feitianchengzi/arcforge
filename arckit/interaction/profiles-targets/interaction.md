# Profiles Targets - 交互规范

## 交互策略

### 核心任务

用户维护当前 Skill 项目的配置组，并把选定配置组应用到 Agent 用户级目录、项目内 Agent 目录或自定义目标目录，同时通过漂移检查确认目标状态。

### 主路径

用户先在配置组列表中选择或编辑配置组，保存后进入目标区域。用户选择 Agent 目标和可选项目目录，或添加自定义目标，确认解析后的目标路径，执行漂移检查或应用操作。应用完成后系统展示复制与跳过结果，并刷新漂移报告。

### 决策点

用户在全部技能和部分技能之间决策，在 Agent 用户级部署、Agent 项目级部署和自定义目录之间决策，并根据漂移报告决定是否再次应用或人工处理目标差异。

### 信息揭示

配置组编辑优先展示配置组列表和当前草稿。目标区域优先展示目标历史与漂移结果，再展示当前来源、配置组、目标路径和主操作。

### 状态流

配置组页面初始展示当前配置组草稿。保存中保持草稿可见。保存成功后刷新工作区快照。目标页面未检查时显示漂移空状态。漂移检查成功后显示技能/资产差异。应用成功后显示复制跳过摘要并刷新漂移。错误状态保留表单输入和目标路径。

### 反馈与恢复

配置保存失败时保留草稿。目录选择取消时保留原路径。应用失败时保留目标路径和配置组选择。目标已存在目录时系统跳过而不覆盖，并通过结果和漂移报告提示。

### 输入输出边界

输入包括配置组名称、描述、技能选择、Agent 目标标识、项目目录、自定义目录和操作触发。输出包括保存后的配置、解析后的目标路径、应用结果、目标历史、漂移报告和完整差异窗口。

## 页面状态

| 状态 | 触发条件 | data-kit 关键控件 |
|------|---------|-----------------|
| 加载中 | 保存配置、应用配置组或检查漂移 | ProgressView / Form |
| 成功（内容） | 配置组和目标数据可用 | Form / Picker(segmented) / List |
| 空状态 | 无目标历史或尚未检查漂移 | ContentUnavailableView |
| 错误 | 保存失败、配置组不存在或目标写入失败 | ContentUnavailableView + 重试 |

## 导航流程

### 进入方式

用户从项目页签进入 Profiles 或 Destinations。两个区域共享当前项目快照。

### 页面内导航

用户在配置组列表和目标表单之间切换焦点。完整漂移通过独立窗口打开。

### 离开方式

用户切换到审计、共享或其他项目。未保存草稿不写入配置文件。

## 交互行为

### 配置组列表

- **点击配置组:** 系统展示该配置组草稿。
- **点击 New profile:** 系统创建唯一名称的草稿配置组并选中。
- **点击 Delete profile:** 系统删除当前草稿；只剩一个配置组时按钮不可用。

### 配置组表单

- **勾选 All skills:** 系统选中全部技能并禁用单项技能选择。
- **取消 All skills:** 系统允许逐项选择技能。
- **点击 Save profiles:** 系统保存配置、重新扫描并保持当前配置组。
- **点击 Cancel:** 系统恢复为来源配置。

### 目标应用

- **勾选 Agent:** 系统把该 Agent 加入用户级部署目标；若已选择项目目录，则解析为项目内 Agent 技能目录。
- **添加项目目录:** 系统打开支持隐藏文件夹的目录选择器；取消时保留旧路径。
- **添加自定义目标:** 系统打开支持隐藏文件夹的目录选择器，并按所选目录原样加入目标。
- **校验目标组合:** 系统要求至少存在一个 Agent 目标或一个自定义目标；存在项目目录但没有 Agent 时，在 Agent 区域提示必须选择 Agent。
- **点击 Check drift:** 系统生成漂移报告。
- **点击 Apply:** 系统复制缺失技能和共享资产，跳过已存在目录，并刷新漂移报告。

### 完整差异

- **点击 View full diff:** 系统打开只读差异窗口。

## 弹窗/对话框

| 名称 | 触发 | data-kit | 内容 |
|------|------|----------|------|
| Drift Diff | 点击 View full diff | Window | 配置组、目标目录、条目状态和文件级哈希 |
| Select directory | 添加项目目录或自定义目标 | Sheet | 支持隐藏文件夹的系统目录选择器 |

## 错误处理

| 错误类型 | 展示方式 | 恢复操作 |
|---------|---------|---------|
| 配置组不存在 | 页面错误状态 | 选择存在配置组或保存配置 |
| 保存失败 | 顶部状态或表单错误状态 | 保留草稿并重新保存 |
| 没有 Agent 或自定义目标 | 目标区域提示并禁用应用 | 选择 Agent 或添加自定义目标 |
| 项目目录没有 Agent | Agent 区域提示 | 选择至少一个 Agent 或移除项目目录 |
| 目标路径为空 | 按钮禁用 | 选择 Agent、自定义目录或项目目录组合 |
| 写入失败 | 应用区域错误状态 | 修改目标路径后重试 |
| 哈希读取失败 | 漂移区域错误状态 | 检查目标文件权限后重试 |

## 加载策略

| 场景 | 策略 |
|------|------|
| 保存配置组 | 表单保留，操作按钮进入处理中语义 |
| 应用配置组 | 目标路径和配置组保持可见 |
| 检查漂移 | 漂移区域显示加载占位，完成后替换列表 |
