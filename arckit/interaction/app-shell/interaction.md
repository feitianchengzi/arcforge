# App Shell - 交互规范

## 交互策略

### 核心任务

用户在应用外壳中选择或下载一个 Skill 项目，并在固定项目上下文中进入扫描、审计、配置组、目标应用和共享任务。

### 主路径

用户打开应用后从侧边栏选择最近项目，系统扫描该项目并展示项目健康状态。用户也可以通过添加项目弹窗打开本地目录或下载 GitHub 来源，下载或选择完成后进入同一扫描路径。

### 决策点

用户在本地目录和 GitHub 来源之间选择添加方式。系统通过项目名称、技能数量、审计评分、下载状态和错误状态帮助用户判断当前项目是否可继续操作。

### 信息揭示

侧边栏优先展示项目列表和项目状态。顶部工具栏优先展示当前项目、扫描状态、Git 环境状态和重新扫描入口。项目内页签只在存在有效快照后展示。

### 状态流

初始状态没有当前项目。用户选择项目后进入扫描中状态。扫描成功后进入内容状态并展示页签。GitHub 来源下载失败或扫描失败时进入错误状态。最近项目为空或项目无效时保持空状态。

### 反馈与恢复

扫描、初始化、下载和环境检测通过顶部状态或项目列表反馈。下载失败保留失败项目记录，用户可以移除后重新添加。扫描失败保留当前路径，用户可以重新扫描、初始化配置或切换项目。

### 输入输出边界

输入包括本地目录、GitHub/Git 来源、重新扫描触发和初始化配置触发。输出为工作区快照、环境状态、最近项目记录和错误信息。下载中或失败项目不触发工作区扫描。

## 页面状态

| 状态 | 触发条件 | data-kit 关键控件 |
|------|---------|-----------------|
| 加载中 | 打开项目或重新扫描 | AppLayout / Sidebar / ProgressView |
| 成功（内容） | 扫描完成且存在工作区快照 | AppLayout / Sidebar / ProjectHeader / stage-scroll |
| 空状态 | 未选择项目或最近项目为空 | ContentUnavailableView / Sheet |
| 错误 | 下载失败、扫描失败或 Git 不可用 | ContentUnavailableView / Alert |

## 导航流程

### 进入方式

应用启动进入外壳。用户从本地目录选择器、GitHub 来源下载或最近项目列表进入具体项目。

### 页面内导航

用户通过项目页签在 Overview、Skills、Profiles、Destinations、Share 和 Audit 之间切换。页签切换不改变当前项目。

### 离开方式

用户可以移除当前项目记录，系统回到未选择项目状态。用户也可以切换到另一个最近项目。

## 交互行为

### 侧边栏项目列表

- **点击项目:** 系统打开项目并扫描工作区。
- **点击下载中项目:** 系统展示下载中状态，不执行扫描。
- **点击失败项目:** 系统展示失败详情和恢复入口，不执行扫描。
- **点击移除:** 系统移除最近项目记录；若移除当前项目，主内容回到空状态。

### 顶部工具栏

- **点击 Rescan:** 系统重新扫描当前项目并刷新快照。
- **点击 Init config:** 系统写入默认配置后重新扫描。

### 添加项目弹窗

- **查看当前来源:** 当弹窗用于编辑项目来源时，系统在入口选择上方展示当前保存的来源类型和来源地址或本地路径。
- **点击 Local folder:** 系统打开目录选择器。
- **点击 Download:** 系统下载 GitHub/Git 来源到本地缓存，成功后打开下载项目。

## 弹窗/对话框

| 名称 | 触发 | data-kit | 内容 |
|------|------|----------|------|
| Add Skill project | 点击添加项目 | Sheet | 本地目录入口、GitHub 来源输入、示例结构 |
| Edit project source | 点击编辑项目来源 | Sheet | 当前已配置来源、本地目录入口、GitHub 来源输入 |
| Settings | 点击设置 | Sheet | 语言选择 |
| Git unavailable | 环境检测失败 | Alert | Git 缺失提示和受影响能力 |

## 错误处理

| 错误类型 | 展示方式 | 恢复操作 |
|---------|---------|---------|
| 目录选择取消 | 顶部状态文本 | 重新点击添加项目 |
| 下载失败 | 侧边栏失败项目和主内容错误状态 | 删除失败记录后重新下载 |
| 扫描失败 | 顶部错误状态 | 重新扫描、初始化配置或切换项目 |
| Git 不可用 | 顶部环境提示 | 安装 Git 后重启或重新检测 |

## 加载策略

| 场景 | 策略 |
|------|------|
| 打开项目 | 顶部显示扫描中，项目上下文立即切换 |
| 下载来源 | 侧边栏创建下载中临时项目 |
| 重新扫描 | 保留当前页签并刷新快照 |
